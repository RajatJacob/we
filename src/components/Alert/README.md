# Alert

A component to display an alert.

## Properties

* ` title ` : The alert's title.(Optional)
* ` message ` : The alert message.
* `type` : A string that describes the type of alert. The accepted types are:
  + ` default ` (Gray)
  + ` success ` (Green)
  + ` info ` (Blue)
  + ` warning ` (Yellow)
  + ` danger ` (Red)
